#  🛤️ 34n0.github.io

### Installation

1. Clone this repository:

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
npm install
```

### Development

To start a local development server:

```bash
npm run dev
```
To preview a production build:

```bash
npm run preview
```

### Deployment

To build for production:

```bash
npm run build
```
